## TCP/IP 协议

* 网络数据传输通过数据包的形式传输，容易出错和丢失；
* ip 协议负责把数据送到目的主机, UDP 把数据包送到对应应用程序，TCP 把数据完整送到 应用程序；
* UDP & TCP 区别， 各自特点，应用场景；
